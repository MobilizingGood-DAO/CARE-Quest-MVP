CARE QUEST

CARE QUEST is a lightweight transparency dashboard for builder incentive programs. The idea came from a simple observation. Most programs only highlight the final winners, but very little of the actual building process is visible while it is happening. CARE QUEST explores a simple alternative by giving builders a place to publish commitments, share proof of progress, and show what they are working on throughout a program.

In this MVP, participating projects can create a profile, describe what they intend to deliver, and attach updates such as GitHub links, demos, or milestone progress as they continue building. These updates appear in a shared dashboard so organizers, mentors, and the broader community can follow along and see how projects evolve over time.

CARE QUEST also encourages collaboration within the ecosystem. Builders can strengthen their standing by helping other teams, sharing knowledge, or contributing to shared public goods within the Avalanche community. These contributions can improve a builder’s overall score and visibility, reinforcing the idea that strong ecosystems grow through cooperation as well as competition.

The goal is not to add complexity or new financial systems. CARE QUEST focuses on improving visibility and coordination. By making commitments and progress easier to track, programs can better understand participation, encourage accountability, and showcase the work happening inside their ecosystem.

This prototype was built as part of Avalanche Build Games using a simple modern web stack. The interface was designed using v0.dev to quickly prototype the dashboard experience, and the codebase is managed in GitHub and iterated on through Cursor. The current version is intentionally lightweight and off chain, focused on validating the core workflow of commitments, progress updates, and transparent program visibility.

If the model proves useful, the same infrastructure could later support stronger verification, reputation tracking, or optional integrations with on chain systems. For now, CARE QUEST is exploring whether making the building process visible can make incentive programs more transparent and engaging for everyone involved.
